⚙ Configuração Supabase

Cole as credenciais do seu projeto Supabase abaixo.
Elas ficam salvas no localStorage do seu navegador.

1 · Execute este SQL no SQL Editor do Supabase:

-- Execute no SQL Editor → New query create table public.visits ( id uuid default gen_random_uuid() primary key, place_key text not null, country text not null, country_code text, state text, display_name text, lat float8 not null, lng float8 not null, visit_date date, notes text, photo_urls jsonb default '[]'::jsonb, created_at timestamptz default now() ); alter table public.visits enable row level security; create policy "public_access" on public.visits for all using (true) with check (true);

2 · Em Storage → New bucket, crie um bucket chamado travel-photos e marque como público.
